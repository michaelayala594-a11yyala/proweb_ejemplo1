¿Qué es un lenguaje de programación?

Es un conjunto de instrucciones y reglas sintácticas que permiten comunicarse con una computadora para crear programas, aplicaciones y sistemas. Facilita la lógica que el equipo debe seguir para ejecutar tareas específicas.

¿Cuál es el comando para crear una rama en git?

El comando es:git branch nombre_de_la_rama. Este crea una nueva rama sin cambiarte a ella automáticamente.

¿Qué comando de Git se debe ejecutar para confirmar que la nueva rama ha sido creada y que el repositorio se ha movido a ella?

Debes usar: git checkout nombre_de_la_rama. Esto cambia el contexto de trabajo a la nueva rama. También puedes usar git switch nombre_de_la_ramaen versiones modernas de Git.

¿Por qué es necesaria la ejecución de git push para que el compañero pueda ver la respuesta en la rama de GitHub?

Porque git push envía los cambios locales al repositorio remoto en GitHub. Sin este paso, los demás colaboradores no pueden ver ni acceder a las actualizaciones realizadas en tu entorno local.

¿Cuáles son los dos pasos principales que debes realizar dentro de GitHub para que tu compañero obtenga los permisos necesarios para contribuir y ejecutar su primer git push en tu proyecto?

Primero, debes agregarlo como colaborador en la configuración del repositorio. Segundo, él debe clonar el repositorio con su cuenta y configurar correctamente su entorno Git para poder hacer git push a las ramas que tenga permiso.
